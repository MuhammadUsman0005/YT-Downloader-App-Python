# YT-Downloader-App-Python
# 🎥 YouTube Downloader Pro (Desktop App)

A fast, lightweight, and powerful desktop application to download YouTube videos, full playlists, audio, and thumbnails. Built with **Python** and **Tkinter**, powered by the robust `yt-dlp` engine.

## 🚀 Features

*   **📺 Single Video Download:** Support for multiple resolutions (360p, 720p, 1080p, 4K).
*   **📑 Full Playlist Support:** Download entire playlists with one click (Auto-creates folders).
*   **🎵 Audio Extraction:** Convert videos to high-quality MP3 directly.
*   **🖼️ Thumbnail Saver:** Download video thumbnails in HD.
*   **⚡ Real-Time Progress:** Live progress bar, download speed, and file size tracking.
*   **📂 Custom Save Location:** Choose where to save your files.

---

## 🛠️ Prerequisites

Before running the app, ensure you have the following installed:

1.  **Python 3.8+**: [Download Here](https://www.python.org/downloads/)
2.  **FFmpeg**: Required for merging high-quality video/audio and MP3 conversion.
    *   *Windows:* Download `ffmpeg.exe` and place it in the project folder.
    *   *Mac/Linux:* Install via terminal (`brew install ffmpeg` or `sudo apt install ffmpeg`).
